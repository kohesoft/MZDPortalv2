Menu yüklemek için aşağıdaki formatı kullanarak menü fotoğrafınızı yükleyebilirsiniz:
// Fotoğrafın adını "menu_YYYYMMDD.jpg" formatında belirleyin.
// Örnek:

menu_20250526.jpg

Mola yüklemek için aşağıdaki formatı kullanarak mola fotoğrafınızı yükleyebilirsiniz:
// Fotoğrafın adını "break_schedue_*.jpg" formatında belirleyin.

// break_schedule_*.jpg formatında belirleyin.

// Örnek:
break_schedule_Merkez.jpg


// Örnek:
break_schedule_genel
